# JS
The Propel WordPress Framework's JavaScript currently supports ES12 along with jsx for use with Gutenberg editor scripts.

jQuery can still be used but is strongly discouraged in favor of vanilla JavaScript.

[ESLint](/docs/best-practices/linting/js-eslint.md) should be configured and used to lint all JS source files.
